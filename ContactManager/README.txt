Instructions to run the program :

on eclipse : 
import the project.
open src -> Manager -> Right click on ContactInterface->run as java application.

on UTD CS machines
transfer the folder into the CS machine
Change Directory to ContactManager/bin/Manager
run the program - > Java Manager.ContactInterface 

Features added / changed :
Window is now Maximize-able and re-sizable.
Status bar is now a label. No borders.
Shows Green colored success messages.
Shows Red colored Failure messages.
Added new fields with length restrictions.
Added Validation of user input.


Features Not supported :
Started off with absolute layout. Maximise does not make the form centered.
